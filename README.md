# Full-Stack Express Application

A complete full-stack web application with user authentication, CRUD operations, and responsive design.

## Features

- 🔐 **User Authentication** - Register, Login, Logout using Passport.js
- 📝 **CRUD Operations** - Create, Read, Update, Delete resources
- 🗄️ **MongoDB Database** - Data persistence with Mongoose ODM
- 🎨 **TailwindCSS** - Modern, responsive UI design
- 📱 **Mobile Responsive** - Works on all devices

## Tech Stack

| Technology | Purpose |
|------------|---------|
| Node.js | Backend runtime |
| Express.js | Web framework |
| MongoDB | Database |
| Mongoose | ODM for MongoDB |
| EJS | Templating engine |
| TailwindCSS | Styling |
| Passport.js | Authentication |
| bcryptjs | Password hashing |
